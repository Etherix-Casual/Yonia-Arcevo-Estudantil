📈 O que é uma Progressão Aritmética (PA)?

Uma Progressão Aritmética é uma sequência de números onde cada termo, a partir do segundo, é igual ao anterior somado a uma constante chamada razão (r). É como dar passos de tamanho fixo em uma reta numérica.

a₁ primeiro termo
r razão (passo)
aₙ termo geral
✨ Exemplo clássico:
(2, 4, 6, 8, 10, ...)
Aqui, a₁ = 2 e r = 2 (cada termo aumenta 2 unidades). Esse tipo é chamado de PA crescente.

📐 Fórmulas Essenciais

🔹 Termo Geral

aₙ = a₁ + (n - 1) · r

Permite encontrar qualquer termo da PA conhecendo o primeiro termo, a razão e a posição n.

🔹 Soma dos n primeiros termos

Sₙ = (a₁ + aₙ) · n / 2

Fórmula de Gauss: soma do primeiro com o último, multiplicado pelo número de termos e dividido por 2.

🔍 Classificação da PA

📈 Crescente (r > 0)

Ex: (3, 5, 7, 9, ...)
a₁ = 3, r = 2

📉 Decrescente (r < 0)

Ex: (10, 7, 4, 1, ...)
a₁ = 10, r = -3

⚖️ Constante (r = 0)

Ex: (5, 5, 5, 5, ...)
todos os termos são iguais

💡 Dica: Para achar a razão, subtraia qualquer termo pelo anterior: r = a₂ – a₁ = a₃ – a₂ = ...

🧠 Passo a passo: como resolver

Exemplo 1: Encontrar o 10º termo

Dados

PA: (3, 7, 11, 15, ...) → a₁ = 3, r = 4, queremos a₁₀.

Fórmula

aₙ = a₁ + (n – 1)·r

a₁₀ = 3 + (10 – 1)·4 = 3 + 9·4 = 3 + 36 = 39

Resposta

O décimo termo é 39.

Exemplo 2: Somar os 20 primeiros termos

PA

(5, 9, 13, 17, ...) → a₁ = 5, r = 4, n = 20.

Passo 1: a₂₀

a₂₀ = 5 + (20 – 1)·4 = 5 + 76 = 81

Passo 2: Soma

S₂₀ = (a₁ + a₂₀)·20 / 2 = (5 + 81)·10 = 86·10 = 860

💎 Dicas de Ouro

🧩
Razão negativa?

Se r < 0, a PA é decrescente. Ex: 10, 7, 4, 1, ...

🎯
Três termos consecutivos

Em uma PA, o termo central é a média aritmética dos vizinhos: x, y, z → y = (x+z)/2

⚡
Fórmula da soma alternativa

Sₙ = n·(2a₁ + (n-1)r)/2 (útil quando não conhece aₙ)

📌
Interpolação aritmética

Inserir meios aritméticos entre dois números: calcule r = (b - a)/(k+1).
